# SQL Server Self-Study Scripts

This repository contains SQL Server scripts written during my self-study journey while following Baraa's 30-hour SQL Server course. The repository is meant to serve as a structured revision space for concepts, queries, syntax patterns, and hands-on practice scripts that I worked through during the course.

## About this repository

The learning reference for this repository is Baraa's SQL Server course and its accompanying original scripts. Those original materials were the starting point for my learning process.

However, the scripts in this repository are not just copied course files. They are my own working study scripts built while learning from that material, with additional structure and annotations added for personal understanding and revision.

## What is different in these scripts

Compared to the original course scripts, my versions include:

- Personal comments written in my own words
- Timestamps mapped to the course for easier backtracking
- Better internal structure across the full script
- Cleaner organization for revision
- Extra notes explaining why a query works
- Small reminders for common mistakes or confusing points

These additions make the scripts much more useful for future revision than raw lecture notes alone.

## Why this repo exists

The main purpose of this repository is revision and long-term retention.

While Baraa's notes and original scripts are still useful as a reference, this repository is designed to be more practical for repeated review. The comments and restructuring make it easier to revisit concepts later without needing to rewatch the entire course every time.
